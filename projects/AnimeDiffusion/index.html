Abstract

Being an essential stage in cartoon animation creation pipeline, colorizing anime line drawing images are usually done manually, which is tedious and time-consuming. Reference-based line drawing colorization is an active research area in computer graphics, in which different methods, mainly based on generative adversarial networks (GAN), have been developed to automate this process. The task is challenging that relies on precise cross-domain long-range dependency modelling between line drawings and reference images. Because of this, existing GAN-based methods still cannot generate high quality results comparable to manual colored results. In this paper, a new method called AnimeDiffusion is proposed, based on diffusion models, for automatic colorization of anime face line drawings. To the best of our knowledge, this is the first attempt to adopt diffusion model in reference-based line drawing colorization task. For the colorization task, a hybrid training strategy is designed, including a pre-training stage for diffusion model with classifier-free guidance and a fine-tuning stage for quickly updating color tone with a target reference colored image. After the model is trained, colorization is done through inferencing the model with a new color reference by updating the fine-tuning stage. Utilizing this hybrid training strategy, our intensive diffusion model can produce astonishing colorization results that exhibit semantic coherence and maintain the intricacies of line drawings within few sampling steps. Since there is no existing high-resolution anime face dataset specialized for colorization method evaluation, an anime face line drawing colorization benchmark dataset, containing 31,696 training data and 579 testing data, is prepared and shared. Extensive experiments and a user study have demonstrated that AnimeDiffusion outperforms state-of-the-art GAN-based models, both quantitatively and qualitatively, for anime face line drawing colorization.
